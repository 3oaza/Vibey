## 🚀 Overview
**Vibey** is a high-performance browser extension built for artists, designers, and visual thinkers. It transforms your browser into a dynamic workspace where you can collect, arrange, and annotate ideas from any website. Whether you're working on character designs, UI mood boards, or visual research, Vibey streamlines your creative process.

---

## 🌟 Key Features

### 🖼️ Infinite Canvas
* **Fluid Navigation**: Infinite pan and zoom (up to 400%) for a seamless workspace.
* **Instant Collection**: Simply drag any image from a webpage and drop it onto your canvas.
* **Smart Queue**: Collect multiple images into a "Holding Queue" before deciding where to place them.

### 🖌️ Creative Suite
* **Annotation Tools**: Draw, sketch, and highlight directly on your mood board with a professional brush engine.
* **Typography**: Add notes and labels with custom fonts, colors, and responsive text layers.
* **Advanced Layering**: A full layer management system (Reorder, Hide/Show, and Grouping).

### 📤 High-Fidelity Exporting
* **Static**: PNG, JPEG, and WebP support with high-resolution output.
* **Animated**: Export your canvas sessions as smooth **High-Res GIFs** or **4K MP4** videos (perfect for Behance or social media).
* **Match View**: Choose to export your entire project or just your current viewport.

---

## 🛠️ Technology Stack
- **Frontend Core**: Vanilla JavaScript (ES6+), HTML5 Canvas, CSS3 Grid/Flexbox.
- **Engines**: 
  - `CCapture.js` for frame-perfect recording.
  - `gif.js` & `gifenc.js` for fast & high-quality GIF generation.
- **Compliance**: Fully Manifest V3 compliant.

## 📦 Installation Guide

### 🌐 For Chrome, Edge, and Brave
1. Download the repository source code.
2. Open your browser and go to `chrome://extensions/`.
3. Enable **Developer Mode** (usually a toggle in the top-right corner).
4. Click **Load Unpacked** and select the `Vibey_organized` folder.

### 🦊 For Firefox
1. Open Firefox and type `about:debugging` in the address bar.
2. Select **This Firefox** from the left menu.
3. Click **Load Temporary Add-on...**.
4. Navigate to the project folder and select the `manifest.json` file.
> *Note: Temporary add-ons in Firefox will stay active until you restart the browser.*
